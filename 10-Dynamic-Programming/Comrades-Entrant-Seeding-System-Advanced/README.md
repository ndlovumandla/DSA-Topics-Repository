# Comrades Entrant Seeding System (Advanced)

Difficulty: Advanced  
Topic: Dynamic Programming Optimization

## Storyline
Yusuf optimizes Comrades entrant seeding under batch-capacity constraints so congestion pressure is minimized. The assignment has overlapping subproblems and cannot rely on greedy choices alone. This advanced implementation provides top-down memoization, bottom-up tabulation, allocation reconstruction, and a built-in equivalent check for correctness.

## What This Program Demonstrates
- Top-down memoized DP
- Bottom-up tabulated DP
- Decision reconstruction from DP states
- Cache and overlapping subproblem reduction
- Built-in equivalent check
- Edge cases: empty entrants, zero capacity, duplicate benefits
