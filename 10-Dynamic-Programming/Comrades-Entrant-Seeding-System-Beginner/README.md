# Comrades Entrant Seeding System (Beginner)

Difficulty: Beginner  
Topic: Dynamic Programming

## Storyline
Yusuf helps the Comrades seeding committee allocate entrants to batches while respecting capacity and minimizing congestion pressure. A naive recursive search recomputes many identical partial allocations. Dynamic programming solves this by caching subproblems (top-down memoization) or filling a DP table once (bottom-up tabulation).

## What This Program Demonstrates
- Top-down memoization
- Bottom-up tabulation
- Overlapping subproblems and optimal substructure
- Cache usage to avoid recomputation
- Edge cases: empty input, zero capacity, single entrant
