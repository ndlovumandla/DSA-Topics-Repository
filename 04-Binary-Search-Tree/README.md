# Binary Search Tree Topic

This folder contains binary search tree implementations and exercises that demonstrate ordered insertion, search, recursion, and traversal.

## Subprojects
- Kruger National Park animal census (beginner): Kruger-National-Park-Animal-Census
- National Library of South Africa catalogue (intermediate): National-Library-of-South-Africa-Catalogue
- Kruger National Park advanced species registry (advanced): Kruger-National-Park-Animal-Census-Advanced
