- a topic-level README index

## How To Run Any Project

1. Open the project folder.
2. Compile the Java file.
3. Run the compiled class.

General pattern:

```powershell
cd "<topic-folder>\<project-folder>"
javac <MainClass>.java
java <MainClass>
```
