# Metrorail Cape Town Network

Difficulty: Beginner  
Topic: Graph Traversal (BFS and DFS)

## Storyline
Ravi models Cape Town Metrorail as a graph where stations are vertices and rail links are undirected edges. The journey planner must quickly answer whether a route exists and which stations are visited. BFS is used for shortest-stop routes, while DFS is used for deep exploration of connected lines.

## What This Program Demonstrates
- Adjacency list graph from scratch
- BFS shortest-stop path reconstruction
- DFS exploration order
- Visited set logic
- Queue (BFS) and stack (DFS) built from scratch
- Edge cases: unknown station, disconnected station, same source and destination

## Run
